# MovieCatalog

## Описание

MovieCatalog — SPA-приложение на Angular для просмотра списка фильмов и детальной информации о каждом фильме.  
Данные загружаются с сервера на основе параметров маршрута.

Приложение демонстрирует работу с роутингом, HTTP-запросами и управлением состояниями загрузки.

## Функциональность

- просмотр списка фильмов
- переход на страницу фильма по id
- загрузка данных фильма с сервера
- обработка состояний загрузки:
  - loading
  - success
  - not-found
  - error
- возврат к списку фильмов

## Технологии

- Angular
- TypeScript
- RxJS
- Angular Router
- HttpClient

## Архитектура

- `MovieService` отвечает за получение данных с API
- компоненты отвечают только за отображение и управление состоянием
- данные передаются в шаблон через `Observable` и `async pipe`
- состояние экрана контролируется через явные статусы (`status`)

## Установка и запуск

Установить зависимости:

```bash
npm install
```

Запустить json-server:

```bash
npx json-server db.json --watch --port 3000
```

Запустить Angular-приложение:

```bash
ng serve
```

## Маршруты

- / — список фильмов
- /movies/:id — страница конкретного фильма

## Статусы загрузки

- loading — выполняется запрос к серверу
- success — данные успешно получены
- not-found — фильм с указанным id не найден (404)
- error — ошибка сервера или сети
